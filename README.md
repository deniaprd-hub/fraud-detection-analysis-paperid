# fraud-detection-analysis-paperid

# 🛡️ Digital Payment Fraud Detection - PT Paper ID

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk mendeteksi indikasi penipuan dan kolusi dalam transaksi digital menggunakan dataset dari PT Paper ID.

## 🛠️ Tools yang Digunakan
- **SQL (SQLite):** Untuk ekstraksi data dan identifikasi anomali.
- **Python:** Untuk analisis jaringan (Network Analysis).
- **Tableau:** Untuk pembuatan dashboard pemantauan real-time.

## 📊 Hasil Analisis
- Berhasil mengidentifikasi transaksi anomali di luar jam operasional.
- Mendeteksi pola kolusi antara Buyer dan Seller dengan volume transaksi tidak wajar.
