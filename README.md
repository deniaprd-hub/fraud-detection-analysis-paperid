# fraud-detection-analysis-paperid

1. Business Problem
Fraud detection diperlukan untuk mengurangi kerugian akibat transaksi ilegal

2. Objective
Mengidentifikasi transaksi fraud berdasarkan pola data

3. Key Insight 
Transaksi fraud lebih sering terjadi pada nominal kecil tapi frekuensi tinggi
Waktu transaksi tertentu lebih berisiko
Pola tertentu menunjukkan anomali

5. Model / Analysis (singkat saja)
Menggunakan Random Forest / Logistic Regression untuk klasifikasi

5. Recommendation
Implementasi sistem alert otomatis
Threshold transaksi mencurigakan
